# Los 14 alérgenos de declaración obligatoria (EFSA / Reglamento UE 1169/2011)

Base para poblar la tabla `allergens`. Slugs en snake_case, estables (no
renombrar una vez usados en `food_allergens`).

| slug | nombre |
|------|--------|
| `gluten_cereals` | Cereales que contienen gluten (trigo, centeno, cebada, avena, espelta, kamut) |
| `crustaceans` | Crustáceos |
| `eggs` | Huevos |
| `fish` | Pescado |
| `peanuts` | Cacahuetes |
| `soybeans` | Soja |
| `milk` | Leche y derivados (incluida lactosa) |
| `tree_nuts` | Frutos de cáscara (almendra, avellana, nuez, anacardo, pistacho...) |
| `celery` | Apio |
| `mustard` | Mostaza |
| `sesame` | Granos de sésamo |
| `sulphites` | Dióxido de azufre y sulfitos (>10 mg/kg) |
| `lupin` | Altramuces |
| `molluscs` | Moluscos |
